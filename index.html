FlowRun Privacy Policy

Last updated: April 3, 2026

1. Overview

FlowRun is a Chrome extension that automates AI image batch generation on Google Flow (labs.google/fx).

2. Data Collection

FlowRun does NOT collect, store, or transmit any personal information.

FlowRun does not operate any external servers.

3. Data Processing

Prompts: Processed only within the browser and sent directly to Google Flow.

Reference Images: Stored temporarily in browser session memory (RAM) and deleted when the tab is closed.

Settings: Saved in Chrome local storage (chrome.storage.local) and never leave the device.

Downloads: Saved to the Chrome download folder specified by the user.

4. Third-Party Sharing

FlowRun does NOT share any data with third parties.

5. Permissions

activeTab: To inject prompts into the Google Flow page.

sidePanel: To provide the user interface.

storage: To save user settings locally.

downloads: To auto-download generated images.

6. Google Flow

FlowRun interacts with Google Flow (labs.google/fx). For Google data handling, see Google Privacy Policy.

7. Children

FlowRun does not knowingly collect data from children under 13.

8. Changes

This policy may be updated. Changes will be posted on this page.

9. Contact

Questions? Open an issue at GitHub Issues.
