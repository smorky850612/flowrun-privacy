FlowRun 개인정보처리방침

Privacy Policy | 최종 수정일: 2026년 4월 3일

1. 개요

FlowRun("본 확장 프로그램")은 Google Flow(labs.google/fx)에서 AI 이미지 배치 생성을 자동화하는 Chrome 확장 프로그램입니다. 본 개인정보처리방침은 FlowRun이 사용자 데이터를 어떻게 처리하는지 설명합니다.

2. 수집하는 개인정보

FlowRun은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

FlowRun은 외부 서버를 운영하지 않으며, 사용자의 데이터를 어떠한 서버로도 전송하지 않습니다.

3. 데이터 처리 방식

프롬프트 텍스트: 사용자가 입력한 프롬프트는 브라우저 내에서만 처리되며, Google Flow 웹사이트에 직접 전달됩니다. FlowRun의 외부 서버로는 전송되지 않습니다.

레퍼런스 이미지: 사용자가 업로드한 레퍼런스 이미지는 브라우저 세션 메모리(RAM)에만 임시 저장되며, 브라우저 탭을 닫으면 즉시 삭제됩니다. 디스크에 영구 저장되지 않습니다.

설정 데이터: 딜레이, 자동 다운로드 여부 등 사용자 설정은 Chrome의 로컬 스토리지(chrome.storage.local)에 저장됩니다. 이 데이터는 사용자의 기기를 벗어나지 않습니다.

다운로드 파일: 자동 다운로드된 이미지는 사용자가 지정한 Chrome 다운로드 폴더에 저장되며, FlowRun은 이 파일에 대한 접근 권한을 보유하지 않습니다.

4. 제3자 데이터 공유

FlowRun은 어떠한 사용자 데이터도 제3자와 공유하지 않습니다.

광고 네트워크, 분석 도구, 외부 API 등과의 데이터 공유가 없습니다.

5. 권한 사용 목적

activeTab: 현재 탭의 Google Flow 페이지에 프롬프트를 자동 입력하기 위해 사용됩니다.

sidePanel: 사용자 인터페이스(프롬프트 입력, 설정, 레퍼런스 관리)를 제공하기 위해 사용됩니다.

storage: 사용자 설정(딜레이, 자동 다운로드 등)을 로컬에 저장하기 위해 사용됩니다.

downloads: 생성된 이미지를 자동으로 다운로드하기 위해 사용됩니다.

6. Google Flow 연동

FlowRun은 Google Flow(labs.google/fx) 웹페이지의 DOM과 상호작용합니다. Google Flow에서의 데이터 처리에 대해서는 Google 개인정보처리방침을 참조하세요.

7. 아동 개인정보 보호

FlowRun은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 변경 사항

본 개인정보처리방침은 변경될 수 있으며, 변경 시 이 페이지에 업데이트됩니다.

9. 문의
