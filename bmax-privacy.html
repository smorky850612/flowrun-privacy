AuraX Bmax — 개인정보처리방침

최종 수정일: 2026년 5월 2일  |  시행일: 2026년 5월 2일

1. 개요

AuraX Bmax("확장 프로그램")는 Gemini AI 콘텐츠 생성, Google Flow 이미지 생성, 네이버 블로그 자동 포스팅 기능을 제공하는 Chrome 확장 프로그램입니다. 본 방침은 확장 프로그램이 수집·처리·보관·공유하는 데이터와 사용자 권리를 설명합니다.

2. 확장 프로그램이 사용자에게 입력을 요청하는 데이터

AuraX Bmax는 서비스 이용을 위해 아래 항목을 사용자에게 직접 입력하도록 요청합니다. 각 항목의 저장 위치와 서버 전송 여부를 명확히 안내합니다.

입력 요청 항목 입력 위치 저장 위치 서버 전송 여부
Google 계정 (이름, 이메일, 프로필 사진) Google OAuth 로그인 화면 AuraX 인증 서버 + 브라우저 로컬 전송함 — 사용자 인증 목적
Gemini API 키 설정 화면 → API 키 입력란 브라우저 로컬 스토리지 (chrome.storage.sync) 전송 안 함 — 로컬에서 Gemini API 호출 시에만 사용
네이버 로그인 ID 설정 화면 → 네이버 계정 입력란 브라우저 로컬 스토리지 (chrome.storage.sync) 전송 안 함 — 네이버 블로그 자동 로그인에만 사용
네이버 로그인 비밀번호 설정 화면 → 네이버 계정 입력란 브라우저 로컬 스토리지 (chrome.storage.local) 전송 안 함 — 네이버 블로그 자동 로그인에만 사용
네이버 블로그 ID 설정 화면 → 블로그 설정 브라우저 로컬 스토리지 (chrome.storage.sync) 전송 안 함 — 포스팅 대상 블로그 식별에만 사용
Cloudflare Worker URL 설정 화면 → 서버 설정 브라우저 로컬 스토리지 (chrome.storage.sync) 전송 안 함 — 인증 서버 주소 로컬 저장
블로그 포스팅 키워드 메인 화면 → 키워드 입력란 세션 중에만 사용 (저장 안 함) Google Gemini API로 전송 — 콘텐츠 생성 목적
블로그 작성 지침 (글쓰기 지침 텍스트) 설정 화면 → 나만의 지침 AuraX 서버 (익명 저장) 전송함 — 서비스 품질 개선 목적, 개인 식별 정보 미포함
블로그 작성 지침 수집 안내: 사용자가 지침을 저장할 때 해당 텍스트가 서비스 품질 개선 목적으로 AuraX 서버에 익명으로 저장됩니다. 이름·이메일 등 개인 식별 정보는 포함되지 않습니다. 이에 동의하지 않으시면 해당 기능을 사용하지 마세요.

3. 자동으로 수집되는 정보

사용자의 입력 없이 아래 정보가 자동 수집됩니다:

  • Google OAuth 로그인 시: 이름, 이메일 주소, 프로필 사진 URL이 자동으로 수집되어 AuraX 인증 서버로 전송됩니다.
  • 이미지 생성 시: Google Flow에서 생성된 이미지가 자동 다운로드됩니다. 이미지 데이터는 AuraX 서버에 저장되지 않습니다.

4. 개인정보의 처리 목적

  • 인증 및 접근 제어: Google 계정 정보(이름, 이메일)를 AuraX 인증 서버에 전송하여 승인된 사용자인지 확인합니다.
  • 서비스 제공: Gemini API, Google Flow, 네이버 블로그 자동화 기능을 정상적으로 동작시키기 위해 사용자가 입력한 설정값을 로컬에서 처리합니다.
  • 서비스 개선: 사용자가 저장한 블로그 작성 지침(익명)을 분석하여 서비스 품질을 개선합니다.
  • 알림 제공: 작업 완료 또는 오류 발생 시 브라우저 알림을 전송합니다.

5. 개인정보 보유 및 이용 기간

  • Google 계정 정보: 사용자가 탈퇴 또는 삭제를 요청하는 즉시 AuraX 서버에서 삭제합니다.
  • 블로그 작성 지침: 익명으로 보관되며, 삭제 요청 시 즉시 처리합니다.
  • 로컬 데이터: 사용자가 Chrome에서 확장 프로그램을 제거하거나 직접 초기화하면 즉시 삭제됩니다.

6. 개인정보 제3자 제공 및 공유

AuraX Bmax는 수집한 정보를 외부 제3자에게 판매하거나 제공하지 않습니다.

다음의 경우에만 데이터가 전송됩니다:

  • AuraX 인증 서버: Google 계정 정보(이름, 이메일)를 전송하여 사용자 승인 여부를 확인합니다. 이 서버는 제3자가 아닌 AuraX 자체 서버입니다.
  • Google Gemini API: 사용자가 입력한 키워드 및 글쓰기 요청이 전송됩니다. Google 개인정보처리방침 적용.
  • Google Flow: 이미지 생성을 위한 프롬프트가 전송됩니다. Google 개인정보처리방침 적용.
  • 네이버 블로그: 생성된 콘텐츠(제목, 본문, 이미지)가 네이버 블로그 API로 전송됩니다. 네이버 개인정보처리방침 적용.
  • 법령에 따른 제공: 법적 의무가 있는 경우에 한해 관련 기관에 제공될 수 있습니다.

7. 개인정보 보호 조치 (보안)

  • 전송 암호화: AuraX 서버와의 모든 통신은 HTTPS(TLS)로 암호화됩니다.
  • 로컬 격리: Gemini API 키, 네이버 계정 정보 등 민감한 정보는 사용자 기기의 브라우저 로컬 스토리지에만 저장되며, 외부로 전송되지 않습니다.
  • 접근 제한: AuraX 서버에 저장된 사용자 정보는 관리자만 접근할 수 있으며, 승인 처리 목적으로만 사용됩니다.
  • 최소 수집 원칙: 서비스 운영에 필요한 최소한의 정보만 수집합니다.

8. 사용하는 권한 및 목적

  • identity: Google 계정으로 로그인(OAuth 2.0) 처리
  • sidePanel: 확장 프로그램 사이드패널 UI 표시
  • storage: API 키, 설정값을 브라우저 로컬에 저장
  • scripting: Google Flow 및 네이버 블로그 자동화 스크립트 실행
  • downloads: Google Flow에서 생성된 이미지 자동 다운로드
  • clipboardWrite: 생성된 콘텐츠 자동 붙여넣기
  • notifications: 작업 완료·오류 알림 표시
  • debugger: Google Flow 이미지 다운로드 인터셉트 (이미지 저장 자동화)

9. 사용자 권리

사용자는 개인정보와 관련하여 다음의 권리를 행사할 수 있습니다:

  • 열람 요청: AuraX 서버에 저장된 본인의 데이터 목록을 요청할 수 있습니다.
  • 삭제 요청: 계정 탈퇴 및 저장된 모든 데이터의 삭제를 요청할 수 있습니다.
  • 수정 요청: 잘못된 정보의 수정을 요청할 수 있습니다.
  • 처리 정지 요청: 특정 데이터의 처리 중단을 요청할 수 있습니다.

권리 행사는 아래 문의처를 통해 요청하시면 7일 이내에 처리합니다.
