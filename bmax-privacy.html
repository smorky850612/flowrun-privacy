AuraX Bmax — 개인정보처리방침

최종 수정일: 2026년 4월 28일

1. 개요

AuraX Bmax는 Gemini AI 콘텐츠 생성 + Google Flow 이미지 생성 + 네이버 블로그 자동 포스팅 기능을 제공하는 Chrome 확장 프로그램입니다.

2. 수집하는 정보

Google 계정 정보: Google OAuth 로그인 시 이름, 이메일 주소, 프로필 사진 URL을 수집합니다. 이 정보는 서비스 인증 및 접근 제어 목적으로만 사용됩니다.

다음 정보는 사용자 기기 내 브라우저 로컬 스토리지에만 저장되며, AuraX 서버로 전송되지 않습니다:

  • Gemini API 키
  • 네이버 계정 정보 (ID, 비밀번호)
  • 확장 프로그램 설정값

3. 개인정보의 이용 목적

  • 서비스 이용 인증 및 접근 제어
  • 승인된 사용자에게만 기능 제공

4. 보유 및 이용 기간

Google 계정 정보(이름, 이메일)는 사용자 탈퇴 요청 시 즉시 삭제합니다. 브라우저 로컬에 저장된 정보는 사용자가 직접 삭제하거나 확장 프로그램을 제거하면 사라집니다.

5. 제3자 제공

AuraX Bmax는 수집한 정보를 제3자에게 판매하거나 제공하지 않습니다. 법령에서 정한 경우를 제외합니다.

6. 사용하는 권한

  • identity: Google 계정으로 로그인(OAuth) 처리
  • sidePanel: 확장 프로그램 UI 표시
  • storage: 설정값 로컬 저장
  • scripting: Google Flow 및 네이버 블로그 자동화
  • downloads: 생성된 이미지 자동 다운로드
  • clipboardWrite: 콘텐츠 붙여넣기 자동화
  • notifications: 작업 완료 알림
  • debugger: 이미지 다운로드 인터셉트

7. 제3자 서비스

AuraX Bmax는 다음 외부 서비스와 연동됩니다. 각 서비스의 개인정보 처리방침을 함께 확인하시기 바랍니다:

  • Google 개인정보처리방침 (Google Flow, Google OAuth, Gemini API)
  • 네이버 개인정보처리방침 (네이버 블로그)

8. 아동 개인정보 보호

AuraX Bmax는 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다.

9. 방침 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 이 페이지에 최신 내용을 게시합니다.
